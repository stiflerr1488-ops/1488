NP.Maps — Чек‑лист упаковки карточки (1 лист)

Цель: карточка должна быть «понятна за 10 секунд» и вести к действию.

A) Профиль
[ ] Название: без лишних слов, совпадает с вывеской/брендом
[ ] Категории: 1–3 точные (не «всё подряд»)
[ ] Адрес: корректный, метка на карте стоит точно
[ ] Вход/как найти: фото + 1 короткая подсказка (если есть сложности)
[ ] График: актуальный, без «по звонку»
[ ] Телефон: кликабельный и проверен (дозвон)
[ ] Сайт/запись: если есть — ведёт на нужную страницу (не на «пустой» лендинг)
[ ] Мессенджеры: указаны, если вы реально отвечаете быстро
[ ] Атрибуты/особенности: заполнены (оплата, парковка, доставка и т.п.)

B) Описание (оффер)
[ ] Первые 2 строки: что вы делаете + для кого + чем вы отличаетесь
[ ] 1–2 обещания результата без «магии» (не «лучшие», а конкретика)
[ ] Убран «канцелярит» и вода — короткими фразами
[ ] Есть блок «Как выбрать / как мы работаем» (3–5 пунктов)

C) Услуги и цены
[ ] 6–10 ключевых услуг (витрина), без дублей и «прочее»
[ ] У каждой услуги: понятное название + цена «от»/диапазон + 1 уточнение
[ ] Есть пакет/комплекс (если подходит), чтобы повысить средний чек

D) Фото и медиа (главный усилитель доверия)
[ ] Первые 6 фото по логике: 1) вход/фасад 2) команда 3) процесс 4) результат 5) интерьер 6) документы/сертификаты (если уместно)
[ ] Фото светлые, без грязных/тёмных кадров, без размытости
[ ] Единый стиль: не мешаем случайные фото, скрины и мемы вперемешку

E) Отзывы и ответы
[ ] Нет «провалов» по месяцам: есть регулярность
[ ] Негатив не висит без ответа (SLA ответа ≤ 24–48 часов)
[ ] Ответы без конфликта: эмоция → факт → решение → контакт
[ ] Причины негатива собираются в список и закрываются системно

F) Путь к действию (конверсия)
[ ] Есть понятный следующий шаг: звонок / запись / маршрут
[ ] Часы работы совпадают с тем, когда реально отвечают
[ ] Администратор знает 3 частых вопроса и короткие ответы

G) Контроль качества
[ ] Раз в 7–14 дней проверяем: открытия → действия → конверсия
[ ] Фиксируем изменения и причину (что делали именно)
[ ] Если подключаем рекламу — считаем по районам/радиусам и режем слив
