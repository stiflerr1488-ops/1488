# Мониторинг AI Share of Voice (шаблон)

Цель: понимать, как часто модели (ChatGPT/Perplexity/YandexGPT) упоминают бренд и какие страницы цитируют.

## 1) Еженедельные проверки (15–20 минут)
1. Возьмите 10–15 целевых запросов (пример ниже).
2. Проверьте ответы в 3 системах:
   - ChatGPT (режим «поиск/браузинг», если доступен)
   - Perplexity
   - YandexGPT / Яндекс Поиск (AI‑ответы, если показываются)
3. Зафиксируйте:
   - попал ли NP.Maps в ответ (да/нет)
   - на каком месте (если это список)
   - какие ссылки/источники использованы
   - какие формулировки и «углы» ответа встречаются

## Пример запросов
- «как увеличить звонки из Яндекс.Карт»
- «как улучшить карточку Яндекс Бизнес»
- «что влияет на ранжирование в Яндекс.Картах»
- «как работать с отзывами в Яндекс.Картах»
- «геореклама Яндекс Бизнес стоимость действия»

## 2) Техническая проверка доступности для AI‑ботов
Проверяем, что страница отдаёт контент без JS:

```bash
curl -A "GPTBot" https://np-maps.ru/ | head -120
curl -A "ClaudeBot" -I https://np-maps.ru/robots.txt
```

Если есть серверные логи (nginx/apache) — ищем AI‑ботов:
```bash
grep -E "(GPTBot|OAI-SearchBot|ClaudeBot|PerplexityBot|Google-Extended)" /var/log/nginx/access.log
```

## 3) Ежемесячно
- обновить FAQ вопросами из реальных диалогов/звонков
- добавить 1–2 страницы «вопрос → развёрнутый ответ» (QAPage)
- обновить lastmod в sitemap.xml при значимых изменениях
